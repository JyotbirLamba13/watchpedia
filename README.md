# Watchpedia

The world's watch encyclopedia. Explore 37+ brands and 326+ references from watchmakers around the world.

**Live:** [watchpedia.vercel.app](https://watchpedia.vercel.app)

## About

Watchpedia is a comprehensive watch encyclopedia cataloguing luxury and everyday timepieces from brands across the globe. Browse by brand, country, or conglomerate group. Read in-depth articles on watchmaking, movements, complications, and collecting.

## Features

- **326+ Watch References** — Detailed specs, images, and descriptions for hundreds of watches
- **37+ Brands** — From Rolex and Patek Philippe to Seiko and Casio
- **10 Countries** — Swiss, Japanese, German, and more watchmaking traditions
- **Blog & SEO** — 13+ articles with structured data, Open Graph, and internal cross-linking
- **Search** — Full-text search across all watches, brands, and collections
- **Admin Panel** — Manage watches, upload images, and edit data

## Tech Stack

- **Framework:** Next.js (App Router, TypeScript)
- **Styling:** Tailwind CSS v4
- **Database:** Supabase (PostgreSQL + Storage)
- **Deployment:** Vercel
- **Images:** Supabase Storage + CDN
